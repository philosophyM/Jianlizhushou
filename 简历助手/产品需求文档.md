# 简历助手 (ResumeTailor) — 产品需求文档

## 1. 产品概述

### 1.1 产品定位
**简历助手 (ResumeTailor)** 是一款基于 AI 的智能简历定制工具，帮助求职者针对不同公司的招聘要求，生成高匹配度的个性化简历。产品以微信小程序为核心载体，同时支持 Web 端访问。

### 1.2 目标用户
- **核心用户**：应届毕业生、职场 1-5 年经验求职者
- **次级用户**：职场转型者、自由职业者、兼职求职者

### 1.3 核心价值
- **匹配度×效率**：从海投到精准投递，每条简历都为企业量身定制
- **自我认知**：通过科学测评帮助用户找到真正适合的职业方向
- **门槛降低**：无需简历写作经验，粘贴招聘信息即可获得专业简历

## 2. 功能模块

| 模块 | 功能 | 优先级 |
|------|------|--------|
| 首页 | 功能入口、快捷操作、使用引导 | P0 |
| 个人资料 | 基本信息、工作经历、教育背景、技能证书 | P0 |
| 职业测评 | 兴趣测评(霍兰德)、性格测评(大五)、价值观测评、技能评估 | P0 |
| 测评报告 | 职业推荐、性格分析、适合行业、发展建议 | P0 |
| 职位分析 | 粘贴JD、AI解析、关键词提取、匹配度计算 | P0 |
| 简历生成 | AI生成针对性简历、实时编辑、多版本管理 | P0 |
| 简历预览 | 简历渲染、PDF导出、分享 | P1 |
| 历史记录 | 投递记录、版本回溯、数据统计 | P1 |
| 设置 | 偏好设置、API配置、关于我们 | P2 |

---

## 3. 用户流程

+ 注册/登录 → 填写基本资料 → 完成职业测评
+ 获取测评报告 → 了解自我定位
+ 粘贴招聘信息 → AI 解析 JD
+ 选择/优化简历版本 → AI 生成针对性简历
+ 预览/编辑 → 导出/分享

## 4. 数据模型

### 4.1 用户表 (users)
| 字段 | 类型 | 说明 |
| _id | string | 微信 OpenID |
| nickName | string | 昵称 |
| avatar | string | 头像 |
| createdAt | date | 创建时间 |

### 4.2 个人资料表 (profiles)
| 字段 | 类型 | 说明 |
| userId | string | 用户ID |
| name | string | 真实姓名 |
| phone | string | 手机号 |
| email | string | 邮箱 |
| education | object | 最高学历、学校、专业、毕业年份 |
| workYears | number | 工作年限 |
| targetIndustry | string | 目标行业 |
| targetPosition | string | 目标职位 |
| skills | array | 技能列表 [{name, level}] |
| workExperience | array | 工作经历 [{company, title, startDate, endDate, description}] |
| projectExperience | array | 项目经验 [{name, role, description}] |
| selfDescription | string | 自我描述 |

### 4.3 职业测评表 (assessments)
| 字段 | 类型 | 说明 |
| userId | string | 用户ID |
| holland | object | 霍兰德测评结果 {realistic, investigative, artistic, social, enterprising, conventional} |
| bigFive | object | 大五人格结果 {openness, conscientiousness, extraversion, agreeableness, neuroticism} |
| workValues | object | 职业价值观 |
| skills | object | 技能评估 |
| recommendedCareers | array | 推荐职业 |
| createdAt | date | 创建时间 |

### 4.4 职位分析表 (job_analyses)
| 字段 | 类型 | 说明 |
| userId | string | 用户ID |
| rawText | string | 原始招聘信息 |
| parsed | object | 解析结果 {company, position, requirements, responsibilities, keywords} |
| keyRequirements | array | 关键要求 |
| createdAt | date | 创建时间 |

### 4.5 简历表 (resumes)
| 字段 | 类型 | 说明 |
| userId | string | 用户ID |
| jobAnalysisId | string | 关联职位分析ID |
| version | number | 版本号 |
| content | object | 简历内容 |
| targetedText | object | 针对性优化内容 |
| status | string | draft/completed |
| createdAt | date | 创建时间 |

## 5. 非功能需求
- 响应时间：页面切换<1s，AI生成<30s
- 数据安全：用户数据加密存储，支持导出和删除
- 离线能力：基础页面支持离线缓存
- 兼容性：iOS 12+ / Android 8+，微信 8.0+
- 用户隐私：符合《个人信息保护法》
