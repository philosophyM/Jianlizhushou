# 微信小程序上手指南

## 环境准备
1. 安装微信开发者工具
2. 注册小程序账号，获取 AppID
3. 开通云开发环境

## 配置
1. 修改 project.config.json 中的 appid
2. 修改 app.js 中的云环境 ID
3. 上传部署云函数
4. 创建数据库集合：users, profiles, assessments, job_analyses, resumes

## 运行
在微信开发者工具中打开 weapp/ 目录，点击编译。

## 常见问题
- 云函数调用失败：检查部署和数据库集合
- AI 不生效：检查 API Key 配置
- 数据不保存：检查权限设置
